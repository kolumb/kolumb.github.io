Закладки

Полезные программы для графики

FastStone Image Viewer
Быстрый просмотр изображений на весь экран, всех форматов с вожностью настройки яркости, обрезки и т.п.
Blender
Blender — свободный пакет для создания трёхмерной компьютерной графики, включающий в себя средства моделирования, анимации, рендеринга, постобработки видео, создания интерактивных игр.
Inkscape
Inkscape — открытый редактор векторной графики, функционально схожий с Illustrator, Freehand, CorelDraw или Xara X и использующий стандарт W3C под названием Scalable Vector Graphics (SVG). В программе поддерживаются такие возможности SVG как фигуры, контуры, текст, маркеры, клоны, альфа-канал, трансформации, градиенты, текстуры и группировка. Inkscape также поддерживает метаданные Creative Commons, правку узлов, слои, сложные операции с контурами, векторизацию растровой графики, текст по контуру, завёрстанный в фигуру текст, редактирование XML-данных напрямую и многое другое.
yEd Graph editor
Редактор графов.
Graphviz
Визуализация графов. Онлайн версия.
Font Xplorer
Просмотр шрифтов с любым текстом, установка и удаление.

Полезные программы для мультимедиа

VLC
Простой плеер, способный воспроизводить все форматы без установки кодеков.
K-Lite Codec Pack
Сборник кодеков для всех форматов.
BS.Player
BSPlayer - мультимедийный плеер (видео в форматах DivX, HD и AVC, фильмы, аудио, DVD, YouTube). Подходит даже обладателям не очень мощных компьютеров. BSPlayer может воспроизводить любые ролики YouTube непосредственно из Интернета и сохранять их на локальный диск. Оснащен расширенными возможностями поиска и воспроизведения субтитров. Но VLC оказался лучше
Lightworks
Професиональный видеоредактор, который недавно обрёл бесплатную версию.
VSDC
Нетребовательный бесплатный видеоредактор с поддержкой многих форматов и эфектов.
AviSynth
Видеоредактор без интерфейса.
µTorrent
Пока-что небольшой клиент BitTorrent. Альтернатива - qBittorrent
Skype
Или уже пора переходить на Viber?
Audacity
Урок по улучшению аудиозаписи голоса.

Полезные программы для текста

Notepad++
Текстовый редактор со свободным исходным кодом с проверкой орфографии, с простой поддержкой многих языков программирования.
Sublime Text
Быстрый текстовый редактор с очень удобным интерфейсом. Видеообзор
keyla

STDU Viewer
Просмотр электронных книг всех форматов: PDF, EPUB, DJVU...
doPDF
Программа создаёт в системе виртуальный принтер, который используется для печати в формат PDF. В браузере Opera теперь тоже можно печатать в pdf.

Библиотеки, нужные для запуска программ написаных на C++ для Windows.

Библиотеки, нужные для запуска программ написаных на C#.
Java
Приложения, которые написаны на языке программирования Java и доступ к которым можно получить из браузера, называются "апплетами". Крупные компании также используют апплеты Java для интранет-приложений и для систем электронной коммерции.
Total Commander

myDefrag
Дефрагментация диска. Особенность в том, что действительно плотно располагает файлы, несмотря на неперемещаемые папки.

NirSoft

PortableApps

Latency monitor

DPC Latency Checker

Dr WEB CureIt

cCleaner

Revo Uninstaller
Удаляет программы вычищая все остатки из пользовательских и временных папках, реестра и автозагрузки. Удаляет даже проблемные программы.
Unchecky
