Закладки

Полезные программы для графики

FastStone Image Viewer
Быстрый просмотр изображений на весь экран, всех форматов с вожностью настройки яркости, обрезки и т.п.
Blender
Blender — свободный пакет для создания трёхмерной компьютерной графики, включающий в себя средства моделирования, анимации, рендеринга, постобработки видео, создания интерактивных игр.
Inkscape
Inkscape — открытый редактор векторной графики, функционально схожий с Illustrator, Freehand, CorelDraw или Xara X и использующий стандарт W3C под названием Scalable Vector Graphics (SVG). В программе поддерживаются такие возможности SVG как фигуры, контуры, текст, маркеры, клоны, альфа-канал, трансформации, градиенты, текстуры и группировка. Inkscape также поддерживает метаданные Creative Commons, правку узлов, слои, сложные операции с контурами, векторизацию растровой графики, текст по контуру, завёрстанный в фигуру текст, редактирование XML-данных напрямую и многое другое.
yEd Graph editor
Редактор графов.
Graphviz
Визуализация графов. Онлайн версия.
Font Xplorer
Просмотр шрифтов с любым текстом, установка и удаление.

Полезные программы для мультимедиа

K-Lite Codec Pack
Сборник кодеков для всех форматов.
BS.Player
BSPlayer - мультимедийный плеер (видео в форматах DivX, HD и AVC, фильмы, аудио, DVD, YouTube). Подходит даже обладателям не очень мощных компьютеров. BSPlayer может воспроизводить любые ролики YouTube непосредственно из Интернета и сохранять их на локальный диск. Оснащен расширенными возможностями поиска и воспроизведения субтитров.
Download Master
Download Master позволяет значительно повысить скорость закачки файлов через Интернет. Для этого используется разбиение файла на потоки, которые закачиваются одновременно. Программа также поддерживает докачку файла с текущей позиции после обрыва связи.
µTorrent
Пока-что небольшой клиент BitTorrent
Skype
Или уже пора переходить на Viber?

Полезные программы для текста

Notepad++
Текстовый редактор со свободным исходным кодом с проверкой орфографии, с простой поддержкой многих языков программирования.
Sublime Text
Текстовый редактор с неплохой поддержкой многих языков программирования с удобным интерфейсом.
keyla
Настройка смены раскладки клавиатуры на любую клавишу.
STDU Viewer
Просмотр электронных книг всех форматов: PDF, EPUB, DJVU...
doPDF
Программа создаёт в системе виртуальный принтер, который используется для печати в формат PDF. В браузере Opera теперь тоже можно печатать в pdf.

Системные программы

Microsoft Visual C++ Redistributable Package
Библиотеки, нужные для запуска программ написаных на C++ для Windows.
.NET Framework
Библиотеки, нужные для запуска программ написаных на C#.
Java
Приложения, которые написаны на языке программирования Java и доступ к которым можно получить из браузера, называются "апплетами". Крупные компании также используют апплеты Java для интранет-приложений и для систем электронной коммерции.
Total Commander
Замена проводника в Windows.
myDefrag
Дефрагментация диска. Особенность в том, что действительно плотно располагает файлы, несмотря на неперемещаемые папки.
Defraggler
Дефрагментация отдельных файлов. От создателей CCleaner

Диагностика работы компьютера

Aspia
программа для сбора и отображения информации о системе, включая аппаратное обеспечение, программное обеспечение, сетевые параметры. Собранные данные можно сохранять в отчет, который можно открыть на любом компьютере. Aspia – свободное программное обеспечение.
Скачать Teamviewer Quick
Управление компьютером через Интернет. Односторонняя версия без установки.
Windows Sysinternals
Официальные инструменты для Windows, диска, сети и т.д.
CPU-Z
Программа для просмотра характеристик процессора, материнской платы, памяти, видеокарты.
Latency monitor
Разные мониторы и сервисы.
DPC Latency Checker
Программа для просмотра задержки вызовов к материнской плате.
Dr WEB CureIt
Бесплатный кусочек антивируса для полной проверки.
cCleaner
Оптимизация и очистка компьютера.
Revo Uninstaller
Удаляет программы вычищая все остатки из пользовательских и временных папках, реестра и автозагрузки. Удаляет даже проблемные программы.
Unchecky
Автоматически снимает галки при установке бесплатных программ. И предупреждает о смене домашней страницы.
Ninite
Установщик свободных программ в один клик, без рекламных добавок.
