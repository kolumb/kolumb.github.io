Крестики-нолики для изучения таблицы умножения

Цель игры - поставить три крестика или нолика в ряд. Победив на маленьком поле, можно победить на большом, если получится ряд из больших крестика или нолика.


Чтобы поставить отметку, нужно выбрать пару чисел. Например, 2 и 5. Тогда можно поставить отметку на всех десятках.


Второй игрок может переместить один из множителей. Например, нажать на 5, а затем на 2. Получится 2 x 2 = 4. И отметить все четвёрки. Каждый может перемещать только один множитель.
