Лабораторна робота 1

file1.html file2.html

Відповіді до лабораторної роботи.

  1. 1. Відкрийте редактор NotePad. Як це зробити?

    Натиснути клавішу Пуск -> написати "блокнот" -> натиснути Enter.
    Або:
    Пуск -> Програми -> Стандартні -> Блокнот. Я б порадив редактор NotePad++, оскільки він так само швидкий, але підсвічує для зручності теги і краще пряцює з кодуваннями.

  2. 2. Створіть за допомогою текстового редактора html-файл з особистими данними о собі. Який тег використовується для введення тексту?

    body
    До речі, назви тегів не потрібно писати великими літерами.

  3. 3. Задайте назву вікна web-сторінки. Який тег це робить?

    title

  4. 4. Збережіть його під назвою file1.htm у власній папці.

    Розширення htm застаріле, зараз немає обмежень на три літери.
    Тому можна вказувати повне розширення html.

  5. 7. Виконайте форматування тексту у файлі file.html. Які теги використали для форматування?

    b, i, u, h1, center, p, hr

  6. 12. Відкрийте file2.html із розповіддю про себе. Список уподобань створіть як нумерований чи ненумерований список. Чим відрізняється створення цих списків?

    Назвою тегу - ol або ul

  7. 14. Проведіть на сторінці лінії різної ширини, довжини та кольорів. Як це зробити?

    Зазначити параметри width - довжина у пікселях, size - товшина лінії, color - колір лінії.

  8. 15. Виокремти список у тексті іншим шрифтом. Який тег змінює тип шрифта?

    font

На будь-якому сайті можна натиснути Ctrl-U, щоб побачити його html код.
